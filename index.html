ADVINHE A PALAVRA

Como funciona

O objetivo é descobrir qual é a palavra secreta antes que suas vidas acabem. No início da partida, uma palavra é escolhida aleatoriamente e todas as suas letras ficam escondidas. O jogador deve tentar adivinhar a palavra digitando uma letra por vez.

Sempre que uma letra correta for digitada, ela aparecerá em todas as posições em que existir na palavra. Caso a letra não exista, o jogador perderá uma vida e a barra de vidas diminuirá. As letras erradas digitadas também ficarão registradas para evitar repetições. O jogo aceitará somente letras.

A vitória acontece quando todas as letras da palavra forem reveladas. Já a derrota ocorre quando todas as vidas forem perdidas. Ao perder, a palavra correta será mostrada na tela para que o jogador saiba qual era a resposta.
